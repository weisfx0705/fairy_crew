# 電影劇組分組精靈 (Movie Crew Grouping Wizard)

這是一個協助電影系學生進行劇組分組的智慧工具。透過收集學生的技能專長與偏好，並利用 AI 助理 (Google GenAI) 進行客觀且高效的職位分配。

## 🌟 專案簡介

「電影劇組分組精靈」旨在解決劇組分組時的繁瑣與爭議。系統引導使用者完成一連串的設定流程，從名單建立、技能調查到最終的職位分派，讓分組決策更加公開、透明且具備參考依據。

## 🚀 功能特色

本應用程式包含七個主要步驟：

1.  **建立名單 (Roster)**：輸入所有參與分組的學生名單。
2.  **產生技能表單 (Generate Skill Form)**：協助產生詢問學生技能與意願的調查表單。
3.  **上傳技能資料 (Upload Skill CSV)**：匯入收集好的學生技能數據。
4.  **技能總覽 (Skills Overview)**：檢視所有學生的強項與偏好分佈。
5.  **定義劇組 (Define Projects)**：設定將要拍攝的專案、導演人選以及各劇組所需的職位名額。
6.  **上傳最終資料 (Upload Final CSVs)**：確認所有分配參數。
7.  **產生結果 (Results)**：利用 AI 根據上述資料，自動產生最佳的分組建議與理由說明。
